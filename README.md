# Student Survival Hub

Student Survival Hub is a serverless academic planning platform built on AWS.

## Features

- Academic Task Manager
- Exam Survival Planner
- Deadline Panic Meter
- What-If Simulator
- Survival Bot
- LocalStorage support

## Project Track

Serverless Application

## AWS Services

- Amazon S3
- Amazon CloudFront
- Amazon API Gateway
- AWS Lambda
- Amazon DynamoDB
- Amazon CloudWatch
- AWS Certificate Manager
- Amazon Route 53

## Architecture

User → CloudFront → S3 Frontend → API Gateway → Lambda → DynamoDB
